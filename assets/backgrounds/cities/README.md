# City backgrounds (legacy path)

City hero and district artwork lives in the primary pipeline:

**`assets/art/cities/{CityFolder}/`**

See [ART_PIPELINE.md](../../../ART_PIPELINE.md) and use `src/assets/imageRegistry.ts` in code.

This folder exists for compatibility with the UI Art Bible layout diagram.
You may add symlinks or copies here for external tooling — the registry reads from `assets/art/cities/`.
