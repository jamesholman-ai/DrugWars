# Event background images

World event modals and cinematic beats. Suggested naming:

```
{event_type}_{variant}.png
```

Examples:
- `police_crackdown_night.png`
- `supply_shortage.png`
- `cartel_war.png`

Recommended: 1920×1080, 16:9, neo-noir style matching city art bible.

Used by: `EventModal`, future event header art via `uiAssetRegistry`.
