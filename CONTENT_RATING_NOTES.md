# Content Rating Notes — Drug Wars Reloaded

**Developer:** AIventure Studios  
**Website:** https://www.aiventure-studios.com

Use this when completing Apple App Store age rating, Google Play content rating (IARC), and PEGI/ESRB self-assessment questionnaires.

## Game type

- **Fictional strategy / simulation game**
- Single-player, offline
- No user-generated content (UGC)
- No online multiplayer or chat
- No real-money gambling or loot boxes with real currency

## Themes present (fictional)

| Theme | Present | Notes for reviewers |
|-------|---------|-------------------|
| Drug references | Yes | Fictional commodity names used in a trading context; no real-world manufacturing or usage instructions |
| Crime / illegal activity | Yes | Player runs a fictional criminal enterprise; framed as arcade strategy |
| Violence references | Yes | Text-based encounters (police stops, robberies, cartel threats); no graphic depictions |
| Profanity | Minimal / none | Street tone; no strong profanity in core UI |
| Sexual content | No | — |
| Real-money gambling | No | In-game cash is fictional |
| Alcohol / tobacco | No | — |
| Horror | No | — |

## What the game does NOT include

- Real-world instructions for obtaining, using, or selling illegal drugs
- Photorealistic drug imagery
- Graphic violence or gore
- Player-to-player communication
- Social features, forums, or sharing of user content
- In-app purchases (version 1.0.0)
- Advertising (version 1.0.0)
- Location-based features tied to real-world places for illegal activity
